# Task 2: Data Aggregation & Reporting (PostgreSQL)

## Overview
This repository contains Task 2 of the SQL Internship.
The task focuses on data aggregation, analytical reporting, and validation
using GROUP BY, HAVING, and window functions.

## Dataset
Employees Sample Database (Original MySQL Dataset)

Source:
https://github.com/datacharmer/test_db

Note:
The original dataset is designed for MySQL.
All queries in this repository are adapted to PostgreSQL syntax
while preserving the same data model and business logic.

## Tables Used
- employees
- salaries
- departments
- dept_emp

## Key Concepts Covered
- Multi-table joins
- Aggregate functions (COUNT, AVG, MIN, MAX)
- GROUP BY and HAVING
- Window functions (LAG, SUM OVER)
- Salary distribution analysis
- Data validation & quality checks

## How to Run
1. Create tables using scripts in `schema/`
2. Load Employees sample dataset (or logical equivalent)
3. Execute SQL files in `queries/` folder in order
